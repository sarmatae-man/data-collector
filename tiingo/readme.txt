Получение котировок через сервис https://www.tiingo.com/

Необходимо зарегистрироваться на сайте (ник, пароль и почта).
Подтвердить регистацию по ссылке из почты.
Перейти на страницу: https://www.tiingo.com/documentation/end-of-day
Нажать кнопку "Click here to see your API Token" скопировать себе токен.
Скопировать и двойным кликом запустить файл: tiingo_request_GUI.exe
Переключить раскладку клавиатуры на английский язык.
В поле Set API token вставить ( Ctrl + V )  полученный токен.
В поле Set ticker прописать тикер (например: tsla)
Нажать Ок. В каталоге, где расположен файл tiingo_request_GUI.exe,
будет произведена загрузка двух файлов ('adj' с учётом сплитов).
Скопируйте файлы в CSV_Folder
